Password Manager

Your task is to extend the password manager application from the lab part. There will be added support for saving the encrypted user data, storing images and extending the password editor and directory tree. This is a second part of WPF Task for 12 points. In case of any doubts concerning the task, please inspect the sample application or consult your teacher.

Home Part

  • Sample Application
  • Serialization and encription:
    • "Save" menu item creates "Passwords.bin" file in the program directory and stores there the encripted user data (diretories and files). For encription use provided library DataEncription.dll
    • If the "Passwords.bin" file exists, then the password inputted during the login will be used to decrypt data after login
    • If the password is incorrect (DataEncryption.Decrypt method returns null) then the message box with an error is shown
    • If the "Passwords.bin" file does not exist, then the login always succeeds and the password inputted during the login will be used to encrypt data during saving
  • Files and directories tree:
    • All items in the tree have context menu. For all items there is a name of the item, an option to delete and to rename
    • Items can be deleted
    • Items can be renamed. During renaming the item in the tree changes to editable field. Field reverts back after presing the Enter key or after item looses focus
    • All file types and directories can be added to any directory
  • Image file:
    • Adding image file opens image select dialog. The filter in the dialog is set to the supported image types
    • If the dialog is closed, the file is not created
    • Selected image displays in the right side of the application
    • "Save Image" button under the image allows for image saving with save file dialog. Image is saved as PNG and the correct filter in the dialog is set
  • Password generator:
    • "Password Generator" menu opens Password Generator window
    • Window title: "Password Generator"
    • Window size: 400x400
    • Window cannot be resized
    • Window always opens in the center of the main application window
    • When the window is visible, the main application window cannot be selected
    • Font size for all elements is 14
    • Sections separated with a horizontal line
    • Bold font of size 15 for the section names

    • White password with the colored background. Password font is "Consolas". Background color matches password strength
    • Password text can be selected and copied with a mouse
    • Displaying the strength of the password
    • Rounded and transparent "Refresh" and "Copy" buttons
    • "Refresh" generates new password
    • "Copy" copies current password to the clipboard

    • Slider with password length. Value displays to the right of a slider. Minimum is 1, maximum is 128 and default value is 8
    • Checkboxes to include in a password additional symbols: "Upper case letters", "Digits" and "Symbols": !#$%&()*+,-./:;<=>?@[\\]^_{|}~
    • Displaying sliders with numeric values for each checked option
    • Options to use the "Minimum" number of checked symbols or "Exactly" that number of symbols
    • Generating password fitting all the criteria. Default password symbols are lowercase letters
    • Displaying error message instead of a password for options that cannot be fulfilled. The buttons are then hidden
    • Change of any option instantly generates a new password
  • Password list:
    • Passwords are sorted by name and grouped by first letter of a name
    • Group header has gray background and white text
    • Search bar with "Search..." placeholder
    • Searching instantly filters passwords with name containing given string
  • Password viewer and editor
    • Two modes for selected password view: for viewing and for editing
    • Nothing is displayed if there is no password item selected
    • Light gray bar on the bottom of the view
    • The vertical scroll bar for the view appears when the content does not fit in the window

    • Default is view mode
    • Displaying only non-empty password fields: Icon, Name, Email, Login, Password, Password strength progress bar with color corresponding password strength color, Website link, Notes
    • Login and Password use "Consolas" font
    • Password is masked
    • Next to the "Email", "Login" and "Password" there are rounded buttons to copy the field to the clipboard
    • Clicking on email launches system default mailing client
    • Clicking on website launches default browser and navigates to specified website
    • Displaying password "Creation time" and "Last edit time"
    • Rounded button on the bottom of the page to "Edit" and to "Delete" password
    • "Edit" button switches to edit mode
    • "Delete" button removes the password
    • Adding new password switches to edit mode

    • In edit mode the list, adding button and the filter are inactive
    • To the right of the image there are image parameters: Resolution, DPI and Format
    • "Generate" button below the password. It launches Password Generator window with the "Choose" button on the bottom
    • Clicking "Choose" button sets the currently generated password in the password field and closes the window
    • If the generator window is closed then the password is not set
    • Button on the bottom of the view change to "Apply" and "Cancel"
    • "Apply" button updates the password and switches back to view mode. If any changes were made, then the "Last edit time" is updated
    • "Cancel" button switches back to view mode without modifying the password data
  • Tips:
    • BinaryFormatter, MemoryStream, ICommand, DependencyProperty.RegisterAttached, ShowDialog, CollectionViewSource, GroupStyle, ContentControl
